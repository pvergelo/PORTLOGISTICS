# 🚢 Curso Especializado en Logística Portuaria (Colombia)

![Estado del Proyecto](https://img.shields.io/badge/Estado-En%20Desarrollo-blue)
![Licencia](https://img.shields.io/badge/Licencia-MIT-green)

Esta es una plataforma educativa moderna diseñada para estudiantes y profesionales del sector logístico en Colombia. El curso abarca desde el marco normativo (DIMAR, DIAN) hasta las últimas tendencias en **Smart Ports** y sostenibilidad.

## 🌐 Ver la Web en Vivo
🚀 **[Haz clic aquí para ver la plataforma desplegada en GitHub Pages](https://tu-usuario.github.io/tu-repositorio/)**

---

## 📚 Estructura del Programa Académico

El contenido está organizado en 6 módulos técnicos descargables:

1.  **Módulo 1:** Fundamentos del Comercio y Marco Normativo (DIMAR, Supertransporte).
2.  **Módulo 2:** Infraestructura y Operaciones (Grúas STS, RTG, Planos de Estiba).
3.  **Módulo 3:** Logística Internacional e Intermodalidad (Río Magdalena, Fletes).
4.  **Módulo 4:** Protección (PBIP/ISPS) y Ciberseguridad.
5.  **Módulo 5:** Sostenibilidad Ambiental y Cambio Climático.
6.  **Módulo 6:** Gestión del Desempeño (KPIs) y Tecnologías 4.0.
